API Usage
POST requests to run AI flows
1) Start the HTTP bridge
# In PowerShell
PS > node http-server.js
2) POST requests to http://localhost:8765/gemini
curl -X POST http://localhost:8765/gemini \
  -H "Content-Type: application/json" \
  -d '{"prompt":"What is this?"}'
The bridge will forward your request to the active AI provider and return the response.
Notes
- POST body format: {"prompt": "your question"}
- Optional: {"prompt": "...", "attachments": [...]}
- OpenAI-compatible: POST /v1/chat/completions
- Response waits up to 60–180 seconds depending on config
